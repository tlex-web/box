## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-8-05 | Tampering / data loss | deleting all copies of a group | mitigate | Keep-first over sorted order keeps `paths[0]`; pre-flight validates every group retains ≥1 real copy before any deletion; snapshot-tree-unchanged on abort |
| T-8-05-HL | Tampering / data loss | deleting a hardlink alias (frees nothing, destroys a name) | mitigate | Collapse by `(volume_serial, file_index)` via `file_identity`; an alias of the kept inode is excluded from deletion candidates (Pitfall 6); real `std::fs::hard_link` fixture test |
| T-8-05-PARTIAL | Tampering / data loss | partial deletion on a mid-plan error | mitigate | Abort-all-before-any pre-flight (mirrors bulk-rename); deletion only after a clean pre-flight; `?`-propagation stops on first error |
| T-8-05-TOCTOU | Tampering | TOCTOU between identity check and delete | accept | Single-process local CLI; identity computed immediately before the plan; documented accepted (low value, local user context) |
| T-8-05-ANSI | Information Disclosure | `--json` purity / abort-empty-stdout | mitigate | `is_json_on()` fork FIRST; abort `bail!`s to stderr with empty stdout (D-09); `delete_json_plan_and_executed` asserts no `0x1B` |
| T-8-05-DOS | Denial of Service | non-UTF-8 NTFS names under `--json` | mitigate | `to_string_lossy()` (D-4); never `to_str().unwrap()` |
